# SVG Kit

Files in this folder are editable master assets for Inkscape.

- `button_nav_idle.svg`: left-rail button idle state
- `button_nav_active.svg`: left-rail button active state
- `panel_frame.svg`: large overview panel frame
- `status_badge.svg`: compact status chip

All files use live text and plain vector paths so you can edit labels, colors, and stroke widths directly.
